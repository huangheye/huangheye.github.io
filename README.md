# Heye Huang Academic Homepage

This repository is prepared for a GitHub Pages personal website.

## Structure

- `index.html`: homepage entry
- `assets/css/styles.css`: page styles
- `assets/js/script.js`: tab switching logic
- `assets/images/`: place profile photo and future lab images here
- `.nojekyll`: tells GitHub Pages to serve the site as a plain static website

## Deploy To GitHub Pages

1. Create a GitHub repository.
2. If you want the site at `https://YOUR_USERNAME.github.io/`, name the repo `YOUR_USERNAME.github.io`.
3. Upload all files in this folder to the repository root.
4. Push to the `main` branch.
5. In GitHub: `Settings` -> `Pages` -> set source to `Deploy from a branch`.
6. Choose branch `main` and folder `/ (root)`.

## Notes

- This site already uses relative paths, so it can also work in a project repo such as `https://YOUR_USERNAME.github.io/repo-name/`.
- To add a real profile image later, place it in `assets/images/` and replace the placeholder block in `index.html`.
- The `Group` tab is still a scaffold and is intended for the next editing pass.
